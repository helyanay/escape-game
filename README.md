# Escape Game NestJS  
### 4 niveaux pour découvrir les fondations du framework NestJS

Bienvenue agent.  
Votre mission : résoudre les 4 niveaux techniques de cet Escape Game NestJS.  
Chaque niveau est volontairement cassé.  
À vous de comprendre pourquoi… et réparer ce qui doit l’être.

Cet atelier se déroule sous forme de **compétition** entre **binômes** :
le premier duo à valider les 4 niveaux remporte la mission.

# 1. Prérequis

Assurez-vous d’avoir installé :

- **Node.js ≥ 18**  
  Téléchargement : https://nodejs.org/en/

- **npm** (installé automatiquement avec Node)

Vérifiez vos versions :

```bash
node -v
npm -v
```
# 2. Installation du projet
```bash
git clone 
```

Entrez dans le dossier :
```bash
cd escape-game-nestjs
```

Installez les dépendances :
```bash
npm install
```


# 3. Lancer l’application

Pour démarrer le serveur NestJS :
```bash
npm run start:dev
```

L’application sera disponible ici :
```bash
http://localhost:3000
```

# A vous de jouez !
